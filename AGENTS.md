## Project Overview

Build a premium personal portfolio website for a Full Stack Developer with expertise in:

* Flutter Development
* Next.js Development
* Laravel Development
* Firebase
* PostgreSQL
* Computer Vision
* Face Recognition Systems
* Self-Hosted Infrastructure
* DevOps
* AI-Assisted Development

The website should position the owner as a developer capable of building complete products from idea to deployment, monitoring, and maintenance.

---

## Design Principles

Avoid generic developer portfolio patterns.

DO NOT create:

* Skill progress bars
* Circular percentage charts
* Generic hero sections
* Random floating icons
* Generic "Passionate Developer" text
* Template-like layouts

The portfolio should feel like a modern SaaS product website.

Design inspirations:

* Linear
* Vercel
* Stripe
* Raycast
* Resend

Visual characteristics:

* Dark theme
* Clean typography
* Large spacing
* Smooth motion
* Bento grid layouts
* Subtle gradients
* Modern SaaS aesthetics
* Performance-focused animations

---

## Core Narrative

The portfolio must communicate:

"I can build, deploy, secure, monitor, and maintain software products."

Focus on technical achievements rather than listing technologies.

Show problem solving, architecture, infrastructure, and project impact.
